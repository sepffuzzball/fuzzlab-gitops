# Terraform

## Initialize

```sh
# paths assume running from ~/git/terraform/rancher dir
cd ./terraform/rancher
terraform init
```

## Create infrastructure

```sh
# paths assume running from ~/git/terraform/rancher dir
cd ./terraform/rancher
terraform apply
```

## Destroy infrastructure

```sh
# paths assume running from ~/git/terraform/rancher dir
cd ./terraform/rancher
terraform destroy
```
